# Phoenix GPT v3

Phoenix GPT v3 is a ChatGPT-style AI assistant website made with HTML, CSS, JavaScript, and OpenRouter API.

## Features

- ChatGPT-style UI
- Real streaming response
- Multiple chat conversations
- Sidebar chat history
- Thinking animation
- Markdown/code formatting
- Voice input
- Voice output
- Dark/light theme
- Text file upload support
- Copy response
- Regenerate answer
- Clear chat
- Download chat
- Mobile responsive design
- OpenRouter API support

## Setup

1. Create OpenRouter API key.
2. Open website.
3. Click API Key.
4. Paste your API key.
5. Start chatting.

## Deploy

GitHub > Repository Settings > Pages > Deploy from Branch > main > root
